\Spork\View\Helper

\Spork\View\HelperDojo

Provides interface to configure Dojo framework and uses HeadLink and HeadScript view helpers to add framework to application.

Example Configuration array(

'dojo' => array(
    'async' => BOOLEAN,
    'debug' => BOOLEAN,
    'enabled' => BOOLEAN,
    'parseOnLoad' => BOOLEAN,
    'packages' => ARRAY('PACKAGE NAME', ...),
    'src' => 'URL',
)

)
